# Data_221_W26_Assignment_2
10 Question assignment for DATA 221.

The programs created for the following programs are all made for processing different files and webpages for a particular output.

Question 1: identifies duplicate words and counts them.<br>
Question 2: identifies duplicate bigrams and counts them.<br>
Question 3: identifies similar lines of text and records them.<br>
Question 4: made to work only for the included student.csv file, filters for data of students who studies a lot, has internet, and has few absences.<br>
Question 5: made to work only for the included student.csv file, segregates student data by different bands of grade.<br>
Question 6: made to work only for the included crime.csv file, computes unemployment rates for people with a high crime rate and people with a low crime rate.<br>
Question 7: extracts the title and the first paragraph of the main article from the webpage https://en.wikipedia.org/wiki/Data_science.<br>
Question 8: extracts the headings relevant to the content from the webpage https://en.wikipedia.org/wiki/Data_science.<br>
Question 9: unfinished.<br>
Question 10: identifies text lines in files that contain the given keyword.<br>

All code was written using PyCharm 2025.2.5
